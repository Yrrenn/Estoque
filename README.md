##Projeto simples de faculdade

Esse projeto tem o intuito de praticar a linguagem: 
java

Sera feito um sistema simples de estoque com as seguintes funcionalidade:

Adicionar iten, listar iten e a quantidade e  excluir itens.

---------------------------------------------------------------------------
##Opção 1 – Cadastrar Produto
O usuário deverá informar:

Nome do produto

Quantidade inicial em estoque

O produto deverá ser armazenado em vetores. // optei por usar arrays

Exemplo:

Nome do produto: Arroz

Quantidade: 25

 

Produto cadastrado com sucesso!


---------------------------------------------------------------------------

##Opção 2 – Alterar Estoque
O usuário deverá informar:

Nome do produto

Caso o produto exista, informar a nova quantidade em estoque.

Exemplo:

Digite o nome do produto:

Arroz

 

Nova quantidade:

40

 

Estoque atualizado com sucesso!

 

Caso o produto não seja encontrado:

Produto não encontrado.

--------------------------------------------------------------------------- 

##Opção 3 – Mostrar Estoque
Exibir todos os produtos cadastrados.

Exemplo:

===== ESTOQUE ATUAL =====

 

Arroz - 40 unidades

Feijão - 20 unidades

Macarrão - 15 unidades

 

Além disso, o sistema deverá informar:

Produto com maior estoque.

Produto com menor estoque.

Quantidade total de itens armazenados.

Exemplo:

Produto com maior estoque: Arroz (40)

 

Produto com menor estoque: Macarrão (15)

 

Total de itens armazenados: 75

 ---------------------------------------------------------------------------

##Opção 4 – Sair
Encerrar a execução do programa.

Exemplo:

Sistema encerrado.

 ---------------------------------------------------------------------------

##Restrições
Não utilizar:

Banco de dados.

Arquivos.

Métodos próprios.

Classes adicionais.

Utilizar apenas:

Vetores

Estruturas de decisão

Estruturas de repetição

Variáveis

Critérios de Avaliação
Critério
